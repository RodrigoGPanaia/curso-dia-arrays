# curso-dia-arrays

### Curso DIO Estruturas de Repetição e Arrays em Java.

#### Exercicios de Arrays.

1. Crie um vetor de 6 números inteiros e mostre-os na ordem inversa.

2. Faça um Programa que leia um vetor de 6 caracteres, e diga quantas consoantes foram lidas.
   Imprima as consoantes.
   
3. Faça um Programa que leia 20 números inteiros aleatórios (entre 0 e 100) armazene-os num vetor.
   Ao final mostre os números e seus sucessores.
   
4. Gere e imprima uma matriz M 4x4 com valores aleatórios entre 0-9.
